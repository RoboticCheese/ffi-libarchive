ffi-libarchive
==============
A Ruby FFI binding to [libarchive][0].

This library provides Ruby FFI bindings to the well-known
[libarchive library][0]. It aims to be interface compatiable
to libarchive-ruby.

## Why this instead of libarchive-ruby?
